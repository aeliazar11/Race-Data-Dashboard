# Race Data Dashboard

MATLAB dashboard analyzing real Formula 1 lap time data from the
2011 Brazilian Grand Prix. Built to demonstrate race data engineering
skills applicable to NASCAR, IndyCar, and IMSA roles.

## What This Project Analyzes
- Lap time progression and tire degradation curves across a full race
- Driver consistency comparison using statistical distribution analysis
- Cumulative time delta showing race gap evolution lap by lap

## Key Findings
- Vettel (P1) ran a standard deviation of 0.977s vs Button's 1.754s
  demonstrating significantly more consistent pace management
- Delta analysis identifies the exact laps where pit stop strategy
  changed the on-track gap between the two drivers
- Tire degradation clearly visible as gradual lap time increase
  within each stint between pit stops

## Tools
- MATLAB
- Dataset: Kaggle — Formula 1 World Championship 1950-2020

## Skills Demonstrated
- Loading and filtering large datasets (589,000 rows) in MATLAB
- Statistical analysis: median, standard deviation, distribution
- Multi-panel data visualization
- Race engineering data interpretation
